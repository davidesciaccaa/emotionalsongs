Software scritto interamente in Java.
Con l’obiettivo di mettere in pratica le conoscenze acquisite nei corsi di Architettura degli Elaboratori, Programmazione, Algoritmi e Strutture Dati, 
abbiamo progettato l’applicazione Emotional Songs, che è in grado di annotare le emozioni percepite durante l’ascolto di canzoni.  

È da specificare che un utente non loggato ha la possibilità di: 
- effettuare un login;
- registrare un proprio account;
- mostrare un prospetto riassuntivo sui tag emozionali degli utenti (in forma aggregata) per ciascun brano della propria playlist;
- ricercare il titolo di una canzone;
- ricercare un autore.

Invece un utente loggato può annotare le emozioni percepite.
